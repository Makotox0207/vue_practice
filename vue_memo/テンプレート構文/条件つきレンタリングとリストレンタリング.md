# 「条件つきレンタリング」と「リストレンタリング」

## v-ifディレクティブ

```html
<body>
  <script src="vue.js"></script>
  <div id="app">
    <!-- ok: true => OK! -->
    <!-- ok: false, maybeOk: true => maybe OK -->
    <!-- ok: false, maybeOk: false => No...OK -->
    <p v-if="ok">OK!</p>
    <p v-else-if="maybeOk">maybe OK</p>
    <p v-else>No...OK</p>
  </div>
<script>
  new Vue({
    el: '#app',
    data: {
      ok: false,
      maybeOk: true
    }
  })
</script>
  
</body>
```

### テンプレートタグでグループ化

```html
<body>
  <script src="vue.js"></script>
  <div id="app">
    <!-- ボタンを押すと、pタグ3つが消えたり出たりする -->
    <template v-if="ok">
      <p>ニーハオ</p>
      <p>hello</p>
      <p>こんにちは</p>
    </template>
    <button @click="ok = !ok">ボタン</button>
  </div>
<script>
  new Vue({
    el: '#app',
    data: {
      ok: true,
    }
  })
</script>
  
</body>
```
